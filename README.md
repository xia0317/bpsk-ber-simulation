通信系统仿真与性能分析

概述  
对数字调制方案（BPSK、QPSK、16QAM）在AWGN和瑞利衰落信道下的全面蒙特卡洛仿真。包含理论误码率对比、频谱效率分析以及可用于发表的高质量可视化图表。

特性
多调制支持：BPSK、QPSK、16QAM（含格雷编码）
多信道支持：AWGN 和瑞利衰落信道
蒙特卡洛仿真：统计可靠的误码率（BER）估计（至少100次错误）
理论验证：与闭式BER表达式进行对比
全面可视化：四子图的出版级高质量图表
频谱效率分析：功率与带宽效率之间的权衡

关键成果  
验证了QPSK在实现与BPSK相同误码率的同时，频谱效率提高2倍  
量化了16QAM在获得4倍频谱效率时的功率代价（约3.7 dB）  
展示了瑞利衰落环境下性能相较于加性高斯白噪声（AWGN）的退化情况
